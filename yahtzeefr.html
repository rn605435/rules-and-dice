Règles

Pour jouer au jeu :
Le jeu consiste en une série de tours. À chaque tour, vous lancez les dés, puis vous marquez le résultat dans l'une des 13 catégories. Vous devez marquer une fois dans chaque catégorie - ce qui signifie que vers la fin du jeu, vous devrez peut-être vous contenter de zéro dans certaines catégories.

Vous avez cinq dés que vous pouvez lancer, représentés par les faces des dés en haut de la page. Pour commencer, vous lancez tous les dés en cliquant sur le bouton Lancer les dés et voir ce que vous avez obtenu. Le tableau se mettra à jour pour vous montrer votre score de dés actuel dans chaque catégorie disponible. Après avoir lancé tous les dés, vous pouvez soit marquer le lancer actuel, soit relancer tout ou une partie des cinq dés.

Pour garder un dé, appuyez ou cliquez dessus puis cliquez sur le bouton Lancer dés pour changer les autres. Cela relancera les dés, en laissant ceux qui ont été choisis inchangés.

Vous pouvez lancer les dés trois fois au total - le jet initial (dans lequel vous devez lancer tous les dés), plus deux relances d'un ou de tous les dés. Après avoir lancé trois fois, vous devez marquer le score.

Une fois que vous avez marqué le résultat, vous relancez tous les dés et répétez le processus. Vous continuez jusqu'à ce que toutes les 13 catégories soient remplies, moment auquel le jeu est terminé.
